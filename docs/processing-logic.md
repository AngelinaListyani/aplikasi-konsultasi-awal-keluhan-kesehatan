# Processing Logic  

## Document Purpose  
Dokumen ini bertujuan menjelaskan logika pemrosesan data yang dilakukan sistem untuk menghasilkan informasi. 

## Processing Logic Oveerview  
Logika pemrosesan sistem dibangun melalui hubungan logis antar data. Data Gejala dihubungkan dengan Data Relasi Diagnosis yang berperan sebagai Basis Pengetahuan untuk menghasilkan diagnosis. Diagnosis yang dihasilkan menjadi dasar penyajian informasi penyakit yang relevan kepada pengguna.
